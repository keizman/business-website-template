## Quick Start

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

Visit http://localhost:5173 to view the website.

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
